README for Scottish Gaelic Generation for Exhibits
==================================================

This repo serves as the companion to Howcroft et al. 2023:

> David M. Howcroft, William Lamb, Anna Groundwater, and Dimitra Gkatzia. 2023. Building a dual dataset of text- and image-grounded conversations and summarisation in GÃ idhlig (Scottish Gaelic). In _Proc. of the 16th International Conference on Natural Language Generation_. Prague, Czechia. Association for Computational Linguistics.

There are two parts:
* `corpus/` contains the corpus of conversations and summaries in Scottish Gaelic, along with demographic and other metadata; and
* `slurk-server/` contains the modified versions of `slurk` and its bots used for data collection.
